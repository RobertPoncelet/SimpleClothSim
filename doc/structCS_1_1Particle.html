float m_isAnchored
 Whether this particle is "anchored", i.e. the cloth will hang from this point. This attribute is a float rather than a bool so that we know that the next attribute or vector component is always sizeof(float) bytes away.
float m_index
 Where this particle is in the vector. Useful for sending to the shader so it knows where to look in the position/normal textures. It is a float so it can be used in the same vertex buffer as the positions, which are stored as floats too.
float m_mass

Detailed Description
