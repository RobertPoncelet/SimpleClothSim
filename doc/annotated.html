ClothThe main cloth simulation class which is mainly composed of a vector of particles, a vector of springs and a solver
CS::ClothInfoA struct containing the information needed to construct a cloth object
GLWindowThe simulation widget that takes information from the Cloth and draws it to the screen, as well as executing the result of any user input
MainWindowThe main interface that contains the simulation and the UI and links them together
CS::ParticleA structure containing all the attributes of a particle; this is used instead of a class so its members can be accessed by a memory address and an offset
SolverDoes most of the "heavy lifting" of the cloth simulation, i.e. calculating the forces and positions of the cloth components
CS::SpringA struct containing all the attributes of a spring; these connect the particles to form the cloth surface and apply forces to them depending on their length
CS::VertA struct containing a position and common attributes of a vertex; useful for sending to OpenGL
